# Lemmings.NET
Remake of Lemmings in HD

with MonoGame 3.8/NET8.0 (C# VS2022)

F1 = Debug

F12 = Scale * 2 window

F10 = FullScreen

M = Mute/Unmute music

PageUp = Increase master volume

PageDown = Decrease master volume
